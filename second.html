Technologies: React, React-router, Private-route, React-Bootstrap, Node, MongoDB, Firebase, Heroku.

Project-overview:
The Together with Travel agency is a fully responsive website for mobile, laptop and desktop where various tour packages are available. I use the google authentication system, so a user must log in first. A logged-in user can buy products and see previous purchase history, add new service, delete products, update status, and can see all buyers purchase records.
